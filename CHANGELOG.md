# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024

### Adicionado
- Jogo completo de simulação de gestão de Brasília
- Sistema de 4 indicadores (População, Economia, Tecnologia, Meio Ambiente)
- 19 eventos principais com narrativas únicas
- Sistema de eventos em cadeia
- Eventos de longa duração (seca, crise hídrica)
- Menu inicial com opções de jogar e ver créditos
- Sistema de cores com colorama para melhor visualização
- Interface visual com barras de progresso
- Múltiplas condições de Game Over
- Modo software (exibição de valores numéricos)
- Documentação completa do projeto
- README.md em português com instruções detalhadas
- DOCUMENTATION.md com documentação técnica
- requirements.txt com dependências
- .gitignore para Python
- LICENSE (MIT)
- CONTRIBUTING.md com guia de contribuição
- CHANGELOG.md para documentar mudanças

### Recursos Especiais
- Sistema de eventos recursivos
- Eventos que afetam múltiplos indicadores
- Validação de entrada do usuário
- Limpeza de tela multiplataforma
- Função auxiliar print_event() para colorização de texto
